
# 🚀 Sayan Mandal Portfolio

A high-performance, 3D animated personal portfolio built with React, Vite, Three.js, and Framer Motion.

## ⚡ 1-Minute Quick Start

If you have Node.js installed, follow these three steps:

1.  **Prepare Folder**: Move all files from this project into a new folder on your Mac (e.g., `~/Desktop/portfolio`).
2.  **Install Dependencies**: Open Terminal in that folder and run:
    ```bash
    npm install
    ```
3.  **Launch Dev Server**:
    ```bash
    npm run dev
    ```

**Your site is now live at: http://localhost:3000**

---

## 🛠 Tech Stack
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS
- **3D**: React Three Fiber / Drei
- **Animations**: Framer Motion
- **Icons**: Lucide React

## 📸 Personalization
- To change your photo, edit `RESUME_DATA.profileImageUrl` in `constants.tsx`.
- To change your details, update the constants in `constants.tsx`.
